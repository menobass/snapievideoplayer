Examples of how to embed videos from play.3speak.tv

1. Standard Embed (iframe mode)

This is the recommended way to embed videos. The &mode=iframe parameter hides the header and info panel for a clean, embeddable experience.

<iframe src="https://play.3speak.tv/watch?v=meno/1czchhmr&mode=iframe" width="854" height="480" frameborder="0" allowfullscreen> </iframe>

2. Responsive Embed (16:9 aspect ratio)

This maintains a 16:9 aspect ratio and adapts to any screen size.

<div style="position: relative; padding-bottom: 56.25%; height: 0;"> <iframe src="https://play.3speak.tv/watch?v=meno/1czchhmr&mode=iframe" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;" frameborder="0" allowfullscreen> </iframe> </div>

6. üì± Handling Vertical Videos

Philosophy: The 3speak player maintains a consistent 16:9 aspect ratio by default (like YouTube/Twitter). Your frontend controls the container size to handle different video orientations optimally.

Demo Video: vempromundo/hkh2vzzf (Vertical content)

‚ùå Standard Container (Shows Letterboxing)

height="300" /* Standard height */

‚úÖ Taller Container (Better Experience)

height="600" /* Taller for vertical */

CSS Container Strategies

/* Standard container for horizontal/mixed content */ .video-container.standard { max-width: 800px; margin: 20px auto; aspect-ratio: 16 / 9; } /* Vertical container for portrait videos */ .video-container.vertical { max-width: 400px; margin: 20px auto; aspect-ratio: 9 / 16; /* Portrait ratio */ } /* Mobile-responsive */ @media (max-width: 768px) { .video-container.vertical { max-width: 100%; } }

JavaScript Dynamic Detection

// Example: Auto-adjust container based on content async function embedVideo(videoId, containerId) { const container = document.getElementById(containerId); const iframe = document.createElement('iframe'); // You could detect orientation via API or video metadata const isVertical = await detectVerticalVideo(videoId); if (isVertical) { container.className = 'video-container vertical'; iframe.height = '600'; } else { container.className = 'video-container standard'; iframe.height = '300'; } iframe.src = `https://play.3speak.tv/embed?v=${videoId}&mode=iframe`; iframe.width = '100%'; iframe.frameBorder = '0'; iframe.allowFullscreen = true; container.appendChild(iframe); }

Simple JavaScript to auto-embed 3speak URLs:

// Detect and convert 3speak URLs to iframes const videoRegex = /https:\/\/(play\.)?3speak\.tv\/(watch|embed)\?v=([^&\s]+)/g; const content = "Check out: https://play.3speak.tv/watch?v=meno/1czchhmr"; const embeddedContent = content.replace(videoRegex, (match, subdomain, route, videoId) => { return `<iframe src="https://play.3speak.tv/${route}?v=${videoId}&mode=iframe" width="854" height="480" frameborder="0" allowfullscreen></iframe>`; });
